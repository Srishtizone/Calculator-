# Python Calculator

## Overview
This project is a simple calculator developed using Python as part of the CodSoft Python Programming Internship.

## Features
- Performs Addition
- Performs Subtraction
- Performs Multiplication
- Performs Division

## How to Run
1. Run the Python file.
2. Enter two numbers.
3. Choose an operator (+, -, *, /).
4. View the result.

## Technologies Used
- Python
